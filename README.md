# Giới thiệu
Application Quản lý của hàng tiện lợi được xây dựng bằng ngôn ngữ C# trên nền tảng Winform với hệ quản trị cơ sở dữ liệu SQL Server được lấy ý tưởng từ hệ thống POS được sử dụng rộng rãi trong các cửa hàng tiện lợi Circle K, Bách hoá XANH, VinMart,...

## Đề tài
Website bán linh kiện điện tử
